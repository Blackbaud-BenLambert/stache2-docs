To use images in your {{ stache.jsonData.global.productNameShort }} project, add them to your src/assets directory in the source folder.

  • src
    • assets
      • image-name.jpg
If you need to upload a lot of image files, we recommend you organize your images into subfolders by page name.

You can then reference them in your {{ stache.jsonData.global.productNameShort }} pages. The Image component provides a responsive container for the image.

Every image must be referenced with the ~/assets​/ prefix for SKY UX to register it as a static asset and render it.

Image properties

  • imageSource — The path to the image to render.
  • imageAlt — The alt tag for the image. Used for accessibility.
Demo
Code
<stache-image imageAlt="This is an example hero." imageSource="~/assets/image-name.jpg"> </stache-image>

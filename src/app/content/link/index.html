In Angular, relative links for navigation within a site are implemented differently than external HMTL href links. To simplify this process for you, we created a Stache link directive that enables you to use it for external links and internal navigation links within a Stache site, including anchor points on the same page.

Link properties

  • stacheRouterLink — Specifies the link path. To link to a section on the same page, pass in . for the route instead of the full path.

  • fragment — (Optional) Specifies the ID of the anchor point. For Stache page anchors, the ID will be a hypenated, lowercase version of the text.

For external hrefs, stacheRouterLink doesn't enable users to right-click the link and open in a new window or tab. If you have an external link you'd like to open in a new window, or give the users the option open in a new window, use an HTML href instead. Example: I'm an external link. Demo

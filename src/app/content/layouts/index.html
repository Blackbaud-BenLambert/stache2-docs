The sidebar layout is the default {{ stache.jsonData.global.productNameShort }} layout, and adds a side navigation on the left side of your content. The order the sidebar routes appear is determined first by the navOrder attribute on the {{stache.jsonData.global.productNameShort}} tag. If there is no navOrder specified, or if two sibling routes have the same navOrder, the routes will be ordered alphabetically by name.

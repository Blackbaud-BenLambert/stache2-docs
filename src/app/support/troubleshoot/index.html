NPM Install

Scenarios

  • When you were running skyux new --template stache or npm install, did the install time out?

  • Are you having a problem serving your site locally?

  • Did you update the dependencies in the package.json file?

Solution

You may want to re-run npm install. Before you do that, we recommend you delete your entire node_modules folder. In the command line for the Stache site folder, enter npm cache clear. When that completes, enter npm install.
